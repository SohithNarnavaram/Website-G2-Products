# G2 Products - MotoVlogging & Creator Gear Store

India's #1 MotoVlogging & Creator Gear Store - Premium action cameras, gimbals, drones, mics & creator gear.

## Project Overview

G2 Products is an e-commerce platform specializing in premium creator gear including action cameras, gimbals, drones, microphones, and accessories from top brands like GoPro, Insta360, DJI, Rode, and more.

## Technologies Used

This project is built with:

- **Vite** - Build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **React Router** - Client-side routing
- **shadcn-ui** - UI component library
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone https://github.com/SohithNarnavaram/G2-Products.git

# Step 2: Navigate to the project directory
cd G2-Products

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
